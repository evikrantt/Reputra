# Reputra
Privacy-first identity layer using ZK proofs
## ✨ Features

- Zero-knowledge identity verification
- Self-sovereign data control
- On-chain proof system
- Wallet-based authentication

## 🛠️ Tech Stack

- Solidity / Cairo / Rust
- ZK-SNARKs / STARKs
- IPFS / Arweave
- React / Next.js (frontend)
